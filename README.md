# 🚀 Spring Boot Projects

This repository contains a collection of **Spring Boot projects** developed while learning backend development using **Java and Spring Boot**.

Each project demonstrates different backend concepts such as **REST APIs, encryption, URL shortening, task management, and more**.

---

## 📂 Projects in this Repository

### 🔐 Encryption Terminal

A simple backend API that encrypts text using Spring Boot.

📅 Created: **5 March 2025**

Features:

* Encrypt text using API
* Simple frontend interface
* REST API implementation

---

### 🤖 Rover

A backend project demonstrating logic implementation using Spring Boot.

📅 Created: **10 March 2025**

Features:

* Backend logic implementation
* API-based structure

---

### 🔗 URL Shortener

A Spring Boot project that converts **long URLs into short links**.

📅 Created: **13 March 2025**

* REST API implementation

### 📝 Notes API

📅 Created: **21 March 2025**

Features:

* Create notes
* Update notes

* RESTful API structure

### 📋 Task Manager

A backend API for managing tasks.

Features:
